Feasibility Notes
Everything above is CSS-only Leather, wood grain, brass, stitching, engraving — layered gradients plus two inline SVG noise filters. Zero image downloads, zero network weight, works offline.
Ships through your token system Every color derives from existing tokens (parchment, saddle, forest, tier-gold). The materials become ~8 new tokens + 3 Card/Button variants — not 65 hand-styled files.
Where real assets would help Photographic leather close-ups (the mockup's embossed corners) need actual texture images — roughly 30–60 KB each as WebP. CSS gets ~85% of the way; the last 15% is a licensing/asset decision.
Accessibility line Engraved brass text passes contrast only at small sizes with the darkened ink used here. Simple Mode should switch to the flat-token treatment — try the toggle above; that's exactly what it would look like.
